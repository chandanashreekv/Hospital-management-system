# Hospital-management-system


 Key Skills: Java, JDBC, MySQL, OOP, Exception Handling 

 The Hospital Management System, developed using Java, MySQL, and JDBC, provides a
 streamlined solution for managing patient records, appointments, doctors, and inventory.
 By leveraging Java's flexibility, MySQL's reliability, and JDBC for seamless database
 connectivity, it ensures efficient healthcare operations, enhancing both patient care and
 organizational effectiveness.

• Designed, coded, tested, and debugged a console-based Java application supporting CRUD operations, database
connectivity, and appointment scheduling logic
• Implemented a JDBC data access layer for secure, efficient database connectivity and transactions
• Applied modular OOP principles to improve code clarity, maintainability, and scalability
